New

Wormhole will generate a new app w/ empty src/ and test/ directories and a wormhole.js configured to connect to Bitcoin.com’s Cloud.

Usage: new [options] <name>
Options:
-r, --restURL <restURL>          The rest URL to use. default: https://wormholerest.example.com/v1/
-e, --environment <environment>  environment of running BITBOX instance. Ex: production, staging. Default: development
-h, --help                       output usage information

$ wormhole new helloEARTH

wormhole.js

Your wormhole.js file will contain the default settings to connect to Bitcoin.com’s Cloud.

exports.config = {
  networks: {
    development: {
      restURL: 'https://wormholerest.example.com/v1/'
    }
  }
};

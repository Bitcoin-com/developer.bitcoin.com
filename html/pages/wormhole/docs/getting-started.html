Getting Started

Dependencies

NodeJS

NodeJS is a javascript runtime build on Chromeâ€™s V8 engine. npm is the package manager for NodeJS.

Use the official installer from nodejs.org. Install the one which says “Recommended for Most Users”

Command Line

To use commands like new and console:

npm install wormholecash --global

To use inside a command line script or client/server app:

npm install wormholecash --save

Usage

Use the scaffold generator, or manually import Wormole into your code: 

let Wormhole = require('wormholecash/lib/Wormhole').default;
let wormhole = new Wormhole();

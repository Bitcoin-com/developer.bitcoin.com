Tokens and Crowdsales

Create tokens with fixed or managed supply. Issue new tokens in a crowdsale. Revoke units of managed tokens.


// Issue new tokens with manageable supply
Wormhole.PayloadCreation.managed(1, 1, 0, "Companies", "Bitcoin Mining", "Quantum Miner", "", "").then((result) => {
console.log(result);
}, (err) => {
console.log(err);
});

// Issue new tokens in a crowdsale
Wormhole.PayloadCreation.crowdsale(1, 1, 0, "Companies", "Bitcoin Mining", "Quantum Miner", "", "", 2, "100", 1483228800, 30, 2 192978657).then((result) => {
console.log(result);
}, (err) => {
console.log(err);
});

// Revoke units of managed tokens
Wormhole.PayloadCreation.revoke(51, "100").then((result) => {
console.log(result);
}, (err) => {
console.log(err);
});

Data Retrieval

console.log(result);

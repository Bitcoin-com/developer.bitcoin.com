### `detailsByHeight`

Return details about a Block. Lookup the block with a block height.

#### Arguments

- blocks (required):
  - `number`: block height
  - `number[]` of Numbers: block heights

#### Result

- details
  - `Promise<BlockDetails>`
  - `Promise<BlockDetails[]>`

#### Examples

    (async () => {
      try {
        let details = await BITBOX.Block.detailsByHeight(500000);
        console.log(details);
      } catch(error) {
       console.error(error)
      }
